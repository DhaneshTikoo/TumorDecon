TumorDecon/data/sig_matrix_tutorial/README.txt

The datasets located in this directory are used in the TumorDecon signature matrix tutorials 'sig_matrix_tutorial.py' and 'full_tutorial.py'.

They were originally download from NCBI, and can be found at the following links:
